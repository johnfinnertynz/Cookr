# UX Flow

1. Onboarding
   User selects confidence, cooking styles, goals, dietary needs, household size, budget, time, dislikes, appliances, and pantry items.

2. Home Dashboard
   Cookr explains the top recommendation, shows weekly plan slots, and offers primary actions for cooking or grocery list generation.

3. Discovery
   User filters recipes by easy/fast, cheap, high protein, healthy, one-pot, minimal cleanup, meal prep, family-friendly, vegetarian, under 5 ingredients, beginner, NZ supermarket-friendly.

4. Recipe Detail
   User reviews ingredients, estimated cost, servings, difficulty, source/licensing note, and an easier-version suggestion.

5. Meal Plan
   User adds recipes to the current plan. The dashboard previews selected meals and leftovers.

6. Grocery List
   Cookr merges duplicated ingredients, groups by aisle/category, checks pantry staples, estimates basket total, and links each line to Woolworths search.

7. Cooking Mode
   User follows one step at a time, scales servings, checks prep, opens cooking term explanations, and uses panic help for common problems.

8. Learn
   User gets confidence-building lessons and substitutions.
